<h1 align="center">:ribbon: Challenge Amigo Secreto (Alura LATAM)</h1>
<p><small>
  Este proyecto es una aplicación desarrollada como parte del programa Principiante de Programación por Oracle ONE y Alura LATAM con el objetivo de realizar el juego del "Amigo Secreto" de una manera simple y sencilla.
  
  El proyecto permite realizar el sorteo del amigo secreto con los nombre ingresados en el apartado.
</small></p>

<h2>Acceso rápido a la aplicación</h2>

[![Open in GitHub Pages](https://img.shields.io/badge/demo_preview-%23f0bbd9)](https://codebynyto.github.io/Amigo-Secreto/)

<h2>Tecnologías utilizadas:</h2>

✔JavaScript

✔HTML

✔CSS

<h2>Funcionalidades</h2>

• :clipboard: Añadir nombres a una lista.

• :no_entry: Validar que no que agreguen campos vacíos.

• :computer: Visualizar los nombres ingresados en pantalla.

• :twisted_rightwards_arrows: Sortear un nombre de manera aleatoria.

• :star2: Mostrar el resultado en pantalla.
